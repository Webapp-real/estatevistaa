# EstateVista - Premium Real Estate Platform

Next.js 14 + Supabase + Gemini AI

## Setup
1. `npm install`
2. Copy `.env.example` to `.env.local` and add your keys
3. Run SQL from `supabase/schema.sql` in Supabase
4. `npm run dev`
5. Visit `/api/seed` to load demo data

## Deploy
Push to GitHub → Import to Vercel/Netlify → Add env vars

## Features
- Admin panel with AI descriptions
- User dashboard with favorites
- Property search + mortgage calculator
- WhatsApp integration
- Mobile responsive
